```bash
Vite + React + Tailwind v4 + Lucide Icons + LocalStorage = DSA Orbit
```

- **Vite** → Blazing fast dev server  
- **React** → Component-based, reusable magic  
- **Tailwind v4** → Utility-first, zero CSS bloat  
- **Lucide** → Beautiful, customizable icons  
- **LocalStorage** → Your progress, your rules

## Quick Start (3 Commands to Glory)

```bash
# 1. Clone the repo
git clone https://github.com/your-username/dsa-orbit.git
cd dsa-orbit

# 2. Install dependencies
npm install

# 3. Launch into orbit
npm run dev
```

**Open [http://localhost:5173](http://localhost:5173)** → **Prepare to be amazed**
